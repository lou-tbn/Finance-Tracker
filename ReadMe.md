# Personal Finance Tracker

## 🇫🇷 Français

**Description du projet**  
Ce projet est un **projet de cours de M1 Programmation Web**, réalisé par Mohamed YATTASSAYE et Lou TOUBIANA.  
L’application *Personal Finance Tracker* permet aux utilisateurs de suivre leurs revenus, dépenses et objectifs d’épargne.  

**Contenu du repository**  
- `rapport.pdf` : rapport détaillé du projet avec architecture, modèle de données, fonctionnalités et feedback sur le cours.  
- `Creation_BD.sql` : script SQL pour créer la base de données PostgreSQL utilisée, hébergée sur Supabase.  
- `finance-tracker_frontend/` : dossier contenant le code source du frontend Angular.  
- `Finance-Tracker_Backend/` : dossier contenant le code source du backend Spring Boot.  

**Fonctionnalités principales**  
- Gestion des transactions : ajouter, modifier, supprimer  
- Catégorisation des transactions : Food, Rent, Entertainment, etc.  
- Suivi des objectifs d’épargne  
- Dashboard avec statistiques et filtres  

**Technologies utilisées**  
- Backend : Spring Boot (Java)  
- Frontend : Angular  
- Base de données : PostgreSQL via Supabase

## 🇬🇧 English

**Project description**  
This project is a **Master 1 Web Programming course project**, developed by Mohamed YATTASSAYE and Lou TOUBIANA.  
The *Personal Finance Tracker* application allows users to track their income, expenses, and savings goals.  

**Repository contents**  
- `rapport.pdf` : detailed project report including architecture, database model, features, and course feedback.  
- `Creation_BD.sql` : SQL script to create the PostgreSQL database used, hosted on Supabase.  
- `finance-tracker_frontend/` : folder containing the Angular frontend source code.  
- `Finance-Tracker_Backend/` : folder containing the Spring Boot backend source code.  

**Main features**  
- Transaction management: add, edit, delete  
- Transaction categorization: Food, Rent, Entertainment, etc.  
- Savings goals tracking  
- Dashboard with statistics and filters  

**Technologies used**  
- Backend: Spring Boot (Java)  
- Frontend: Angular  
- Database: PostgreSQL via Supabase
